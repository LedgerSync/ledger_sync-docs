# Getting Help

There is a group of passionate maintainers happy to help you get started with LedgerSync.  There are two main
channels for discussing LedgerSync: <a href={socialConfigs.github}>Github</a> and <a href={socialConfigs.slack}>Slack</a>


## Report a bug

Please <a href={`${socialConfigs.github}/issues/new`}>open an issue on Github</a> to report any bugs.  Please check if the bug
has previously been reported and comment on the open issue with your use case.
